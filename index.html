FlexBox

FlexBox adalah suatu mode untuk mengatur tata letak item di dalam sebuah container atau wadah secara horizontal atau vertikal. Item dalam container akan mengembang atau menyusut memenuhi containernya.

Sebelumnya kita buat studi kasus sederhana, membuat tiga buah box didalam sebuah container. 

<div class="container">
    <div class="box1"></div>
    <div class="box2"></div>
    <div class="box3"></div>
</div>

Dan kita akan memberi sedikit css.

.container {
  margin: auto;
  width: 200px;
  border: 3px solid black;
}
.box1 {
  width: 100px;
  height: 100px;
  background-color: salmon;
}              
.box2 {
  width: 100px;
  height: 100px;
  background-color: magenta;
}              
.box3 {
  width: 100px;
  height: 100px;
  background-color: cadetblue;
}

Mengubah container menjadi flexbox

Untuk memberitahu sebuah komponen menjadi container flexbox, kamu hanya perlu memberi properti display: flex, maka komponen itu akan menjadi container dan semua elemen didalamnya menjadi item.

Coba tambahkan properti display: flex pada class container. 

.container {
  display: flex;
}

Setelah container menjadi flex, ketiga item akan berjejer secara horizontal.

flex-direction

Secara default arah dari flex adalah berjejer kesamping / row. Kamu juga bisa mengubah arah flex menjadi berjejer kebawah dengan memberi properti flex-direction: column; pada container. 

.container {
  flex-direction: column;
}

Setelah memberikan flex-direction: column;, maka kotak akan berjejer kebawah. Selain row dan column, kamu juga bisa menggunakan row-reverse dan column-reverse untuk membalik urutan item. 

flex-direction: row | column | row-reverse | column-reverse;

flex-wrap

Secara default semua item akan berjejer kesamping memenuhi containernya hingga ukuran item akan menyusut sampai lebih kecil dari lebar yang diberikan jika container sudah tidak dapat menampung item-itemnya. Untuk menghindari itu, kita bisa menggunakan propreti flex-wrap: wrap; agar item yang lebih kecil dari lebar yang diberikan dapat kembali keukuran awalnya dan item yang tidak tertampung oleh container akan menempati baris berikutnya. 

.container {
  flex-wrap: wrap;
}

flex-flow

Properti ini adalah singkatan dari flex-direction dan flex-wrap. 

flex-direction: row;
flex-wrap: nowrap;

Menjadi:

flex-flow: row nowrap;

justify-content

Secara default, flex item akan berjejer rapat kesamping. Kamu juga bisa mengatur letak item secara horizontal menggunakan properti justify-content: flex-start;.

Kita ubah lebar container sehingga akan terdapat sisa ruang kosong. 

.container {
  width: 500px;
  justify-content: flex-start;
}

Sekarang kita ubah posisi item ke tengah dengan memberikan properti justify-content: center; pada container. 

.container {
  justify-content: center;
}

Ada beberapa nilai dari properti ini yaitu flex-start(default), flex-end, center, space-around, space-between, dan space-evenly. 

justify-content: flex-start | flex-end | center | space-around | space-between | space-evenly

align-items

Selain bisa mengatur item secara horizontal, kamu juga bisa mengatur item secara vertikal menggunakan properti align-items: stretch;.

Tambahkan height: 100px; kemudian hapus height: 100px; pada semua item. Tambahkan juga text pada kotak. 

<div class="container">
    <div class="box1">Kotak 1</div>
    <div class="box2">Kotak 2</div>
    <div class="box3">Kotak 3</div>
</div>
.container {
  width: 500px;
  height: 100px;
  display: flex;
  align-items: stretch;
}
.box1 {
  width: 100px;
  background-color: salmon;
}              
.box2 {
  width: 100px;
  background-color: magenta;
}              
.box3 {
  width: 100px;
  background-color: cadetblue;
}

Secara default nilai align-items: stretch yang artinya item akan memenuhi tinggi dari containernya. Kita bisa memberikan align-items: center; sehingga item akan berada di tengah dan item akan memiliki tinggi sama dengan kontennya. 

.container {
  align-items: center;
}

Ada beberapa nilai dari properti ini yaitu stretch (default), flex-start, flex-end, center, dan baseline. 

align-items: stretch | flex-start | flex-end | center | baseline

align-content

Properti ini memiliki fungsi yang sama dengan align-items, hanya saja properti ini hanya bisa digunakan jika item lebih dari satu baris. 

.container {
  width: 200px;
  flex-wrap: wrap;
  align-content: stretch;
}

Secara default nilai align-content: stretch. Kita bisa memberikan align-content: space-around; sehingga item akan memiliki margin yang sama atas dan bawahnya. 

.container {
  align-content: space-around;
}

Beberapa nilai propertinya yaitu stretch (default), flex-start, flex-end, center, space-around, dan space-between. 

align-content: stretch | flex-start | flex-end | center | space-around | space-between

flex-grow

Properti ini digunakan untuk memperbesar item kesamping memenuhi area kosong container. Secara default flex-grow: 0; yang berarti item tidak dapat membesar atau tumbuh. Flex-grow hanya bernilai 0 dan angka positif tanpa satuan.

Sebelum ditambahkan flex-grow pada item.

Sekarang tambahkan flex-grow: 1; pada kotak 1.

.box1 {
  flex-grow: 1;
}

Maka kotak 1 akan memenuhi area kosong dari containernnya.

flex-shrink

Properti ini digunakan untuk memperkecil item jika container tidak dapat menampung item-itemnya. Secara default flex-shrink: 1; yang berarti item akan mengecil atau menyusut jika container tidak cukup. Jika kamu memberikan properti flex-shrink: 0; pada item, maka item tidak bisa mengecil atau menyusut. flex-shrink memiliki nilai yang sama dengan flex-grow.

Pada gambar diatas, kita beri width: 200px; pada container dan width: 100px; pada semua item. Harusnya item keluar dari containernya karena lebar item melebihi container, tapi karena item memiliki nilai default flex-shrink: 1; maka semua item akan mengecil menyesuaikan lebar containernya.

Sekarang kita beri flex-shrink: 0; pada kotak 1, maka width kotak 1 akan dipertahan untuk tidak berkurang atau mengecil. 

.box1 {
  flex-shrink: 0;
}

Bagaimana jika semua item kita beri flex-shrink: 0;. 

.box1, .box2, .box3 {
  flex-shrink: 0;
}

Maka item akan overflow karena lebar item melebihi containernya.

flex-basis

Properti ini digunakan memberikan nilai lebar pada item. Properti ini mirip dengan width.

Sekarang kita ganti properti width: 100px; pada semua item dengan flex-basis: 100px;. 

.box1, .box2, .box3 {
  flex-basis: 100px;
}

Maka akan mengeluarkan hasil yang sama.

flex

flex merupakan singkatan dari flex-grow, flex-shrink, dan flex-basis. Jika kamu menuliskan satu nilai saja, maka flex sama dengan flex-grow. 

flex-grow: 1;
flex-shrink: 1;
flex-basis: 100px;

Menjadi:

flex: 1 1 100px;

Order

Properti ini digunakan untuk mengatur urutan tata letak dari item, kita bisa memindahkan item yang di tengah menjadi diawal atau item yang ada diawal dipindahkan ke yang paling akhir itu akan sangat mudah denan properti ini. Secara default semua item memiliki nilai order: 0;, sehingga jika kita memberikan properti order: 1; pada salah satu item, maka item itu akan pindah ke akhir. Sebaliknya, jika kita memberikan nilai order: -1; maka item itu akan pindah ke awal.

Sekarang kita pindah kotak 2 berada disebelah kanan.

.box2 {
  order: 1;
}

Nested FlexBox

Nested FlexBox adalah kita membuat flexbox bersarang atau bertumpul dengan kata lain kita menjadi item flexbox menjadi container flexbox juga.

Sekarang kita akan mencoba meletakkan tulisan didalam kotak berada di tengah-tengah. 

<div class="container">
  <div class="box1">
    <p>Kotak 1</p>
  </div>
  <div class="box2">
    <p>Kotak 2</p>
  </div>
  <div class="box3">
    <p>Kotak 3</p>
  </div>
</div>

Sekarang kita tambahkan CSS.

.container {
  width: 300px;
  height: 100px;
  display: flex;
  margin: auto;
  border: 3px solid black;
}

.box1, .box2, .box3 {
  flex: 1;
  display: flex;
  justify-content: center;
  align-items: center;
}

.box1 {
  background-color: salmon;
}

.box2 {
  background-color: magenta;
}

.box3 {
  background-color: cadetblue;
}

.box1 p, .box2 p, .box3 p {
  background-color: aqua;
}
